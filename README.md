# 🧪 Ciencia de Datos en Salud - OpenSaludLab

![Banner del proyecto](https://github.com/opensaludlab/ciencia_datos/blob/main/images/Curso%20Ciencia%20datos.png)

## 📌 Descripción
Este repositorio reúne materiales y proyectos del **Bootcamp de Ciencia de Datos en Salud**, liderado por **OpenSaludLab**. Su propósito es entregar herramientas técnicas y metodológicas para aplicar la ciencia de datos en instituciones de salud pública.

## 🎯 Objetivos
- 📊 Desarrollar conocimientos avanzados en análisis de datos.  
- 🌍 Democratizar el acceso al conocimiento en español.  
- 🏥 Aplicar metodologías de datos para mejorar procesos en salud pública.
