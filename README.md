## 📌 Descripción
Este repositorio reúne materiales y proyectos del **curso de ciencia de datos para deporte**. Su propósito es entregar herramientas técnicas y metodológicas para aplicar la ciencia de datos en entornos deportivos.

## 🎯 Objetivos
- 📊 Desarrollar conocimientos avanzados en análisis de datos.  
- 🌍 Democratizar el acceso al conocimiento en español.  
- 🏥 Aplicar metodologías de datos para mejorar procesos y rendimiento en el deporte.
